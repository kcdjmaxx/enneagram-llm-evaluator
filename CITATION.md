## For Academic Papers

When citing this framework in academic publications, please include:

1. The version number you used (important for reproducibility)
2. A brief description of how you used the framework
3. Any modifications you made to the testing methodology

### Example in-text citation:

*"We assessed LLM personality patterns using the Enneagram LLM Testing Framework (Ross, 2025), running each model through three iterations of both Likert-scale and forced-choice assessments..."*
